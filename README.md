# Weather App

Search any city and get real-time weather conditions.

**Live Demo:** https://weather-o0xlit76g-mur1llo7s-projects.vercel.app/

## Tech stack
React · Vite · OpenWeatherMap API · CSS Modules · Vercel

## Getting started
1. Clone the repo: `git clone: https//:github.com/yourname/weather-app`
2. Install dependencies: `npm install`
3. Add your API key to `.env`: `VITE_WEATHER_API_KEY=your_key`
4. Run locally: `npm run dev`

## Project structure
src/
  components/   UI components (SearchBar, WeatherCard, ErrorMessage)
  hooks/        Custom React hooks (useWeather) 
  services/     API layer (weatherApi.js)
